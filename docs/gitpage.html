How to use It

  1. Click the Run in PWD Playground button above.
  2. Wait for the Docker Playground to load.
  3. Log in to Docker.
  4. Click Start to initialize the session.
  5. Wait for the image to be pulled.
  6. Close the Session Stack Builder tab.
  7. Click the 5173 button on the right side Open Ports.
  8. Wait for the page to load.
